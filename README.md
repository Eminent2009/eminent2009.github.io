# 我的个人网站

![网站预览](https://picsum.photos/id/0/1200/600)

这是一个基于现代前端技术构建的个人网站，采用简约风格设计，用于展示个人技能和联系方式。网站具有响应式布局，可在各种设备上良好显示，并包含丰富的交互效果。

## ✨ 网站特点

- **现代化UI设计**：采用深蓝色为主色调，搭配简洁的布局和清晰的视觉层次
- **响应式布局**：完美适配从手机到桌面的各种设备尺寸
- **交互动效**：包含滚动动画、悬停效果、平滑过渡等增强用户体验的元素
- **完整功能模块**：个人简介、技能展示、项目作品集、联系方式等核心板块
- **项目筛选系统**：支持按类别筛选项目，便于访问者快速找到感兴趣的内容

## 🛠️ 技术栈

- HTML5 语义化标签
- Tailwind CSS v3 样式框架
- Font Awesome 图标库
- JavaScript 交互逻辑
- 纯CSS实现的动画效果

## 📂 页面结构

1. **首页 (`index.html`)**
   - 导航栏（滚动动态效果）
   - 个人介绍区域
   - 技能展示区域（带动画效果）
   - 精选项目展示
   - 联系方式

2. **更多项目页 (`projects.html`)**
   - 项目分类筛选功能
   - 项目卡片网格布局
   - 加载更多项目功能
   - 项目详情展示

## 🌟 特色功能

- 导航栏滚动时的背景变化效果
- 项目卡片悬停时的提升和缩放动画
- 技能条的渐进式加载动画
- 基于类别筛选的项目展示系统
- 响应式移动端菜单（汉堡按钮）
- 使用SVG与CSS改变色彩实现的哔哩哔哩图标（符合B站品牌特征）
